Regras para nomes de identificadores
-podem começar com letras,$ ou _
-não poem começar com números
-podemos usar letras e números
-podemos usar acentos e síbolos
-não podem conter espaços
-não pode conter palavras que o js usa como comando

Obs: maiúsculos e minúsculos fazeem diferença

para trabalhar com os tipos, utilizamos o "typeof"

Ex:
> var n = 200
undefined
> n
200
> typeof n
'number'
> n = 'google'
'google'
> typeof n
'string'
> typeof 6
'number'
> typeof 6.5
'number'
> typeof '6.5'
'string'
> typeof []
'object'
> typeof {}
'object'
> typeof function(){}
'function'
> typeof undefined
'undefined'
> typeof NaN

> typeof Infinity

> typeof null
